# 拖动模式UI控件添加说明

## 需要在Qt Designer中添加到SCModePage（自标定页面）的控件

### 1. 拖动模式控制按钮
- **控件名称**: `dragModeStartBtn`
- **类型**: QPushButton
- **文本**: "进入拖动模式"
- **位置建议**: 在"下一步标定"按钮下方

- **控件名称**: `dragModeStopBtn`
- **类型**: QPushButton
- **文本**: "退出拖动模式"
- **位置建议**: 与"进入拖动模式"按钮并排或下方
- **默认状态**: 禁用（enabled=false）

### 2. 位置记录按钮
- **控件名称**: `dragRecordPos1Btn`
- **类型**: QPushButton
- **文本**: "记录位置1"
- **位置建议**: 在拖动模式按钮下方
- **默认状态**: 禁用（enabled=false）

- **控件名称**: `dragRecordPos2Btn`
- **类型**: QPushButton
- **文本**: "记录位置2"
- **位置建议**: 与"记录位置1"按钮并排或下方
- **默认状态**: 禁用（enabled=false）

### 3. 位置显示标签
- **控件名称**: `dragPos1Label`
- **类型**: QLabel
- **文本**: "位置1: 未记录"
- **位置建议**: 在记录按钮下方

- **控件名称**: `dragPos2Label`
- **类型**: QLabel
- **文本**: "位置2: 未记录"
- **位置建议**: 在位置1标签下方

### 4. 清空按钮（可选）
- **控件名称**: `dragClearBtn`
- **类型**: QPushButton
- **文本**: "清空拖动记录"
- **位置建议**: 在位置显示标签下方

## UI布局建议

```
[开始标定] [停止]
[下一步标定]
[进入拖动模式] [退出拖动模式]
[记录位置1] [记录位置2]
位置1: [显示位置1信息]
位置2: [显示位置2信息]
[清空拖动记录]
[保存数据] [清空数据]
```

## 功能说明

1. **进入拖动模式**: 
   - 发送"SMode"指令，让电机进入单关节点动模式（不发送速度指令时电机自由状态）
   - 激活后可以手动拖动机器人

2. **记录位置1/2**: 
   - 记录当前4个电机的角度值（angle[0-3]）
   - 记录当前姿态值（bodyPose[0-2]）
   - 显示在对应的Label中

3. **退出拖动模式**: 
   - 发送"#Tstop"指令停止运动
   - 禁用记录按钮

4. **清空拖动记录**: 
   - 清空已记录的两个位置数据
   - 仅在拖动模式未激活时可用

## 注意事项

- 代码中已包含安全检查（`if(ui->dragModeStartBtn)`），即使UI文件中没有这些控件，程序也不会崩溃
- 但为了完整功能，建议在Qt Designer中添加上述所有控件
- 控件名称必须与代码中的名称完全一致（区分大小写）
